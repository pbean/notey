# Keyboard shortcuts

Notey is keyboard-first. On macOS, use **`Cmd`** wherever **`Ctrl`** is shown.

## Global (works system-wide)

| Action | Shortcut | Notes |
| --- | --- | --- |
| Summon / dismiss Notey | `Ctrl+Shift+N` | The global capture hotkey. Customizable in onboarding and Settings. Availability depends on the OS compositor. |

## Configurable in-app shortcuts

These act when Notey is focused and can be rebound in **Settings → Shortcuts**
(`Ctrl+,`). Defaults:

| Action | Default |
| --- | --- |
| Command palette | `Ctrl+P` |
| Search notes | `Ctrl+F` |
| New note | `Ctrl+N` |
| Toggle note list | `Ctrl+B` |
| Toggle theme | `Ctrl+Shift+T` |
| Close tab | `Ctrl+W` |

Rebindings must use `Ctrl`/`Cmd` with an optional `Shift`/`Alt` and a letter or
number; conflicting bindings are detected and rejected.

## Reserved shortcuts (not rebindable)

| Action | Shortcut |
| --- | --- |
| Open settings | `Ctrl+,` |
| Next tab | `Ctrl+Tab` |
| Previous tab | `Ctrl+Shift+Tab` |
| Jump to tab 1–9 | `Ctrl+1` … `Ctrl+9` |
| Close overlay / hide window | `Esc` |

## In overlays and lists

| Action | Key |
| --- | --- |
| Move selection | `↑` / `↓` |
| Open / confirm selection | `Enter` |
| Close overlay | `Esc` |
